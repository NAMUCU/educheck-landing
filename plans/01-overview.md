# 01. 랜딩 페이지 개요

## 목적

에듀체크 서비스의 가치를 명확하게 전달하고, 중소형 학원 원장/강사의 무료 체험 신청을 유도하는 랜딩 페이지

---

## 타깃 오디언스

### Primary: 중소형 학원 원장/강사
- 학생 20~50명 규모
- 매일 오답 노트 수작업으로 1~2시간 소요
- 학부모 상담 시 "왜 틀렸는지" 설명에 어려움
- 기존 문제은행 서비스 사용 후 포기 경험

### Secondary: 과외 강사
- 1:1 또는 소규모 그룹 지도
- 학부모 리포트 요청 증가
- 효율적인 학습 관리 도구 필요

---

## 핵심 메시지

### 메인 카피
> "시험지 사진 한 장이 학습 처방전이 됩니다"

### 서브 카피
> "사진만 찍으면 자동 채점, 약점 분석, 학부모 리포트까지"

### 가치 제안 (Value Proposition)
1. **Zero Input**: 오답 데이터 수동 입력 시간 90% 절감
2. **처방 중심**: 단순 분석이 아닌 "다음에 뭘 해야 하는지" 가이드
3. **학부모 소통**: 전문가 수준의 학습 리포트 자동 발송

---

## 전환 목표 (Conversion Goals)

| 우선순위 | 목표 | CTA |
|----------|------|-----|
| 1순위 | 무료 체험 신청 | "2주 무료로 시작하기" |
| 2순위 | 데모 영상 시청 | "30초 데모 보기" |
| 3순위 | 문의하기 | "상담 신청" |

---

## 페이지 구조

```
1. Hero Section
   - 메인 카피 + 서브 카피
   - CTA 버튼
   - 제품 스크린샷/영상

2. Pain Point Section
   - 타깃의 현재 고통 3가지
   - 공감 유도

3. Solution Section
   - 3단계 해결 과정
   - 사진 촬영 → 자동 채점 → 처방 리포트

4. Features Section
   - 핵심 기능 4가지
   - 스크린샷과 함께

5. Social Proof Section
   - 사용 후기
   - 통계 (시간 절감률 등)

6. Pricing Section
   - 요금제 안내
   - 무료 체험 강조

7. FAQ Section
   - 자주 묻는 질문 5-7개

8. Final CTA Section
   - 마지막 전환 유도
```

---

## 성공 지표 (KPIs)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 전환율 | 5% 이상 | 무료 체험 신청 / 방문자 |
| 평균 체류 시간 | 2분 이상 | GA4 |
| 스크롤 깊이 | 70% 이상 | GA4 |
| CTA 클릭률 | 10% 이상 | 버튼 클릭 추적 |
| 이탈률 | 40% 이하 | GA4 |

## 경쟁사 랜딩 페이지 참고

| 서비스 | 강점 | 약점 |
|--------|------|------|
| 매쓰플랫 | 기능 상세 설명 | 너무 복잡, 핵심 가치 불명확 |
| 아이엠스쿨 | 신뢰감 (학교 연계) | B2B 중심, 학원 타깃 아님 |
| 클래스팅 | 깔끔한 UI | 커뮤니케이션 중심, 학습 분석 약함 |

### 차별화 포인트
- **극단적 단순함**: "사진만 찍으면 끝" 강조
- **처방 중심**: 분석 → 처방 연결 시각화
- **학부모 관점**: 리포트 예시 직접 보여주기
