## Section 3: Solution (해결책)

### 섹션 타이틀
```
에듀체크는 다릅니다
사진 한 장이면 끝나니까요
```

### 3단계 프로세스

#### Step 1: 찍기
**타이틀**: 시험지 사진 촬영
**내용**:
```
학생의 시험지를 스마트폰으로 촬영하세요.
5초면 OCR이 답안을 자동으로 인식합니다.
```

#### Step 2: 분석
**타이틀**: 자동 채점 & 약점 분석
**내용**:
```
정답과 오답을 자동으로 채점하고,
어떤 개념이 약한지 즉시 분석합니다.
```

#### Step 3: 처방
**타이틀**: 학습 처방 가이드
**내용**:
```
"이 학생은 완전제곱식이 약합니다.
이 순서로 복습시키세요."
구체적인 처방을 받으세요.
```

**스크린샷**: 학생 상세 - 약점 분석 차트

**타이틀**: "이 순서로 가르치세요"

## Section 7: FAQ

### Q1
**질문**: 어떤 과목을 지원하나요?
**답변**: 현재 수학 과목의 객관식/단답형 문제를 지원합니다. 영어, 국어 과목은 곧 추가될 예정입니다.

### Q2
**질문**: OCR 인식 정확도는 어떻게 되나요?
**답변**: 인쇄체 기준 95% 이상의 정확도를 보장합니다. 혹시 인식이 잘못된 경우 간단히 수정할 수 있습니다.

### Q3
**질문**: 기존 학생 데이터를 옮길 수 있나요?
**답변**: 네, 엑셀 파일로 학생 명단을 한 번에 업로드할 수 있습니다. 기존 데이터 이전도 무료로 도와드립니다.

### Q4
**질문**: 학부모 앱은 어떻게 설치하나요?
**답변**: 학생 등록 시 학부모 전화번호를 입력하면, 앱 설치 안내 문자가 자동 발송됩니다. 앱 없이 카카오톡으로도 리포트를 받을 수 있습니다.

### Q5
**질문**: 해지는 어떻게 하나요?
**답변**: 설정에서 원클릭 해지 가능합니다. 위약금 없으며, 데이터는 30일간 보관 후 안전하게 삭제됩니다.

### Q6
**질문**: 개인정보는 안전한가요?
**답변**: 모든 데이터는 암호화되어 저장되며, 개인정보보호법을 철저히 준수합니다. 학생 정보는 학원 관리자만 접근 가능합니다.
